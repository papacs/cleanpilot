# Security Policy

CleanPilot is a disk cleanup tool, so unsafe deletion behavior is treated as a security-sensitive issue.

Please report issues involving unsafe cleanup targets, unexpected deletion, privilege handling, or command injection risks by opening a GitHub issue with:

- Windows version.
- CleanPilot version or commit.
- Command used.
- Relevant log output.
- Whether the run was dry-run or cleanup mode.

Do not include private file contents or secrets in public reports.
